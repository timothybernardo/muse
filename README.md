## ‚ú® Features

### Albums & Reviews
- Browse new releases and trending albums via Spotify integration
- Rate albums using a 5-star system
- Write and publish detailed reviews
- Like and comment on other users' reviews
- View track listings with integrated lyrics powered by LRCLIB API

### Playlists
- Create custom playlists by adding songs from any album
- Like and comment on playlists
- Browse playlists created by other users

### Social
- Follow other users and see their activity
- View follower/following lists
- Personalized feed showing albums recently reviewed by users

### Discovery
- **New Releases** ‚Äî Fresh albums from Spotify's catalog
- **Recently Reviewed** ‚Äî See what the community is listening to
- **Popular on Muse** ‚Äî Albums with the most listens and reviews
- **Curated Picks** ‚Äî Hand-selected albums by the creator
- **AI-Powered Recommendations** ‚Äî Get personalized suggestions based on your listening history
